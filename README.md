* 主要用于杂七杂八的记录；
* 基于 SpringBoot 2.0.3 + Mapper Mybatis 搭建；
* 含有可独立扩展的基于 Spring Security 的权限模块；
* 任务调度尚未加入；
* 基于 Oracle 分布式锁；
* 手写线程池；
* 基于 Guava 限流；
* 各种加/解密；
* 基于 Hibernate Validator 封装参数校验；
* 基于 Guava 本地缓存；
* 各种工具；

注意：Spring Boot2.0.4必须要Tomcat 8.5以上
